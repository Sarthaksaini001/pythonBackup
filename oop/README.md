# Object-Oriented Programming

This directory contains examples of OOP concepts in Python.

## Topics Covered

- Classes and Objects
- Attributes and Methods
- Constructors and Destructors
- Inheritance
- Polymorphism
- Encapsulation
- Abstraction
- Class Methods and Static Methods
- Properties and Decorators
- Magic Methods

## Example Files

Add your OOP examples here, such as:
- `classes_objects.py` - Basic class examples
- `inheritance.py` - Inheritance examples
- `polymorphism.py` - Polymorphic behavior
- `encapsulation.py` - Data hiding
- `magic_methods.py` - Special methods
- `properties.py` - Property decorators
