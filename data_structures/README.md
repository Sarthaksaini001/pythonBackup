# Data Structures

This directory contains examples and exercises on Python data structures.

## Topics Covered

- Lists and List Comprehensions
- Tuples
- Dictionaries
- Sets
- Strings (Advanced)
- Arrays
- Stacks and Queues
- Linked Lists
- Trees and Graphs

## Example Files

Add your data structure implementations here, such as:
- `lists.py` - Working with lists
- `dictionaries.py` - Dictionary operations
- `tuples.py` - Tuple usage
- `sets.py` - Set operations
- `comprehensions.py` - List/Dict comprehensions
- `stacks.py` - Stack implementation
- `queues.py` - Queue implementation
