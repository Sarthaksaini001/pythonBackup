# Algorithms

This directory contains common algorithm implementations and problem-solving techniques.

## Topics Covered

- Sorting Algorithms (Bubble, Merge, Quick, etc.)
- Searching Algorithms (Binary, Linear)
- Recursion
- Dynamic Programming
- Greedy Algorithms
- Graph Algorithms
- String Algorithms
- Mathematical Algorithms

## Example Files

Add your algorithm implementations here, such as:
- `sorting.py` - Various sorting algorithms
- `searching.py` - Searching techniques
- `recursion.py` - Recursive solutions
- `dynamic_programming.py` - DP problems
- `graph_algorithms.py` - Graph traversal, shortest path
