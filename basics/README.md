# Python Basics

This directory contains fundamental Python programming concepts.

## Topics Covered

- Variables and Data Types
- Operators (Arithmetic, Comparison, Logical)
- Control Flow (if, elif, else)
- Loops (for, while)
- Input/Output
- String Operations
- Basic Functions

## Getting Started

Start with the basics if you're new to Python. Each file focuses on a specific concept with examples and exercises.

## Example Files

Add your Python basics files here, such as:
- `hello_world.py` - Your first Python program
- `variables.py` - Working with variables
- `data_types.py` - Understanding different data types
- `operators.py` - Using operators
- `control_flow.py` - If statements and conditionals
- `loops.py` - For and while loops
- `strings.py` - String manipulation
